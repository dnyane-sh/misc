# Database Server
MongoDB

run 

```mohosh_init```

# Server 
Node.js backend

cd to /server

install dependencies with 
``` npm install ```

use command ``` npm start``` to start server

port: 3001 is used for server

# Client 
React App
    
cd to /client

install dependencies with 
``` npm install ```

use command ``` npm start```

Goto http://localhost:3000 to use the app
